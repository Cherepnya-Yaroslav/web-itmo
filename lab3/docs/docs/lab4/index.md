# Лабораторная работа 4: Клиентская часть на Vue.js

## Описание

В лабораторной работе №4 реализована клиентская часть системы хакатонов из лабораторной работы №3.
Frontend написан на Vue 3 и взаимодействует с REST API на Django REST Framework.

## Цель

- Реализовать web-интерфейсы для работы с сущностями предметной области.
- Подключить аутентификацию и авторизацию через backend API.
- Закрыть сценарии для разных ролей: `admin`, `captain`, `curator`, `jury`.

## Технологии

- Vue 3
- Vue Router 4
- Axios
- Bootstrap 5 (как аналог Vuetify)
- Vite

## Реализованные разделы

- Вход и регистрация пользователей
- Профиль пользователя (обновление учетных данных)
- Список и карточка задач
- Список и карточка команд
- Список и карточка решений
- Раздел оценок для жюри

## Связь с лабораторной №3

Frontend использует backend из `lab3`:

- Базовый URL API: `http://127.0.0.1:8003`
- Токен-аутентификация через Djoser
- Ролевые ограничения контролируются на backend
