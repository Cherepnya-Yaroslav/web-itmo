# Лабораторная работа 1: Работа с сокетами

## Описание

Лабораторная работа посвящена изучению основ сетевого программирования в Python с использованием библиотеки `socket`. Работа состоит из пяти заданий, каждое из которых демонстрирует различные аспекты работы с сетевыми протоколами.

## Цель работы

Овладеть практическими навыками работы с сокетами в Python, изучить протоколы UDP и TCP, научиться создавать простые клиент-серверные приложения и веб-серверы.

Работа состоит из пяти заданий:

## Общие принципы

### Протокол UDP

UDP (User Datagram Protocol) - протокол без установления соединения. Характеристики:

- Быстрая передача данных
- Нет гарантии доставки
- Нет контроля порядка пакетов
- Подходит для простых запрос-ответ сценариев

### Протокол TCP

TCP (Transmission Control Protocol) - протокол с установлением соединения. Характеристики:

- Гарантированная доставка данных
- Контроль порядка пакетов
- Надежное соединение
- Подходит для приложений, требующих надежности

### HTTP протокол

HTTP (HyperText Transfer Protocol) - протокол прикладного уровня для передачи веб-страниц:

- Работает поверх TCP
- Запрос-ответ архитектура
- Методы: GET, POST, PUT, DELETE и др.
- Стандартные заголовки и коды статусов
