# Задание 1: UDP Hello Client-Server

## Описание

Реализация клиентской и серверной части UDP-приложения с использованием библиотеки `socket`. Клиент отправляет серверу сообщение "Hello, server", и оно должно отобразиться на стороне сервера. В ответ сервер отправляет клиенту сообщение "Hello, client", которое должно отобразиться у клиента.

## Требования

- Обязательно использовать библиотеку `socket`
- Реализовать с помощью протокола UDP

## Файлы

- `udp_server.py` - UDP сервер
- `udp_client.py` - UDP клиент

## Как запустить

### 1. Запуск сервера

```bash
cd lab1/task1_udp_hello
python3 udp_server.py
```

Сервер запустится и будет ожидать сообщения на порту 12345.

### 2. Запуск клиента (в другом терминале)

```bash
cd lab1/task1_udp_hello
python3 udp_client.py
```

## Ожидаемое поведение

1. Сервер запускается и ожидает сообщения на порту 12345
2. Клиент отправляет сообщение "Hello, server"
3. Сервер получает сообщение, выводит его на экран и отправляет ответ "Hello, client"
4. Клиент получает ответ и выводит его на экран
5. Оба приложения корректно завершают работу

## Технические детали

### UDP протокол

UDP (User Datagram Protocol) - протокол без установления соединения:

- **Быстрая передача** - нет накладных расходов на установление соединения
- **Нет гарантии доставки** - пакеты могут быть потеряны
- **Нет контроля порядка** - пакеты могут прийти в неправильном порядке
- **Простота** - минимальный overhead

### Основные методы socket для UDP

**Сервер:**
```python
sock = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
sock.bind(('localhost', 12345))
data, addr = sock.recvfrom(1024)
sock.sendto(response, addr)
```

**Клиент:**
```python
sock = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
sock.sendto(message, ('localhost', 12345))
data, addr = sock.recvfrom(1024)
```

## Особенности реализации

1. **Создание сокета** - `socket.SOCK_DGRAM` для UDP
2. **Привязка адреса** - только на сервере через `bind()`
3. **Отправка данных** - `sendto()` с указанием адреса получателя
4. **Получение данных** - `recvfrom()` возвращает данные и адрес отправителя
5. **Кодирование** - работа с байтами через `encode()`/`decode()`
