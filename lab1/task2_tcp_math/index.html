    Задание 2: TCP Math Server

    Описание

    Реализация клиентской и серверной части TCP-приложения для выполнения математических операций. Клиент запрашивает выполнение математической операции, параметры которой вводятся с клавиатуры. Сервер обрабатывает данные и возвращает результат клиенту.

    Доступные операции

    1. Теорема Пифагора - вычисление гипотенузы по двум катетам
    2. Решение квадратного уравнения - нахождение корней уравнения ax² + bx + c = 0
    3. Поиск площади трапеции - вычисление площади по двум основаниям и высоте
    4. Поиск площади параллелограмма - вычисление площади по основанию и высоте

    Требования

    • Обязательно использовать библиотеку socket
    • Реализовать с помощью протокола TCP

    Файлы

    • tcp_server.py - TCP сервер с математическими функциями
    • tcp_client.py - TCP клиент с интерактивным интерфейсом

    Как запустить

    1. Запуск сервера

    cd lab1/task2_tcp_math
python3 tcp_server.py

    Сервер запустится и будет ожидать подключений на порту 12346.

    2. Запуск клиента (в другом терминале)

    cd lab1/task2_tcp_math
python3 tcp_client.py

    Примеры использования

    Теорема Пифагора

    Выберите операцию:
1. Теорема Пифагора

    Технические детали

    TCP протокол

    TCP (Transmission Control Protocol) - протокол с установлением соединения:

    • Гарантированная доставка - все данные доставляются
    • Контроль порядка - данные приходят в правильном порядке
    • Надежное соединение - установление соединения перед передачей
    • Поток данных - данные передаются как поток байтов

    Основные методы socket для TCP

    Сервер: 

    sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
sock.bind(('localhost', 12346))
sock.listen(5)
conn, addr = sock.accept()
data = conn.recv(1024)
conn.sendall(response)
conn.close()

    Клиент: 

    sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
sock.connect(('localhost', 12346))
sock.sendall(message)
data = sock.recv(1024)
sock.close()

    Математические формулы

    Теорема Пифагора

    c = √(a² + b²)
где a, b - катеты, c - гипотенуза

    Квадратное уравнение

    ax² + bx + c = 0
D = b² - 4ac
x1 = (-b + √D) / 2a
x2 = (-b - √D) / 2a

    Площадь трапеции

    S = (a + b) * h / 2
где a, b - основания, h - высота

    Площадь параллелограмма

    S = a * h
где a - основание, h - высота

    Особенности реализации

    1. Установление соединения - TCP требует connect() на клиенте и accept() на сервере
    2. Обработка множественных запросов - сервер может обрабатывать несколько клиентов последовательно
    3. Парсинг данных - клиент отправляет строку, сервер парсит и обрабатывает
    4. Обработка ошибок - валидация входных данных и обработка математических ошибок
    5. Кодирование - работа с байтами через encode()/decode()

    Требования

    • Python 3
    • Библиотека socket (встроенная)
    • Библиотека sys (встроенная)
    • Библиотека math (встроенная)
