    Лабораторная работа 1: Работа с сокетами

    Описание

    Лабораторная работа посвящена изучению основ сетевого программирования в Python с использованием библиотеки socket. Работа состоит из пяти заданий, каждое из которых демонстрирует различные аспекты работы с сетевыми протоколами.

    Цель работы

    Овладеть практическими навыками работы с сокетами в Python, изучить протоколы UDP и TCP, научиться создавать простые клиент-серверные приложения и веб-серверы.

    Структура работы

    Работа состоит из пяти заданий:

    1. UDP Hello Client-Server - базовое UDP приложение
    2. TCP Math Server - TCP сервер для математических операций
    3. HTTP Server - простой HTTP сервер
    4. TCP Chat - многопользовательский чат
    5. Web Server - веб-сервер с GET и POST запросами

    Требования

    • Python 3.6+
    • Библиотека socket (встроенная)
    • Библиотека threading (для задания 4)
    • Библиотека sys, os, datetime, urllib.parse (встроенные)

    Общие принципы

    Протокол UDP

    UDP (User Datagram Protocol) - протокол без установления соединения. Характеристики:

    • Быстрая передача данных
    • Нет гарантии доставки
    • Нет контроля порядка пакетов
    • Подходит для простых запрос-ответ сценариев

    Протокол TCP

    TCP (Transmission Control Protocol) - протокол с установлением соединения. Характеристики:

    • Гарантированная доставка данных
    • Контроль порядка пакетов
    • Надежное соединение
    • Подходит для приложений, требующих надежности

    HTTP протокол

    HTTP (HyperText Transfer Protocol) - протокол прикладного уровня для передачи веб-страниц:

    • Работает поверх TCP
    • Запрос-ответ архитектура
    • Методы: GET, POST, PUT, DELETE и др.
    • Стандартные заголовки и коды статусов

    Оценка работы

    • Выполнение заданий 1-4 и однопользовательского чата (без потоков) — 60% баллов
    • Выполнение заданий 1-5 и многопользовательского чата (с потоками) — 100% баллов

    Задание 1: UDP Hello Client-Server

    Реализация клиентской и серверной части UDP-приложения. Клиент отправляет серверу сообщение "Hello, server", сервер отвечает "Hello, client".

    Задание 2: TCP Math Server

    Реализация TCP-сервера для выполнения математических операций: - Теорема Пифагора - Решение квадратного уравнения - Поиск площади трапеции - Поиск площади параллелограмма

    Подробнее →

    Задание 3: HTTP Server

    Реализация простого HTTP-сервера, который отдает HTML-страницу из файла index.html в ответ на HTTP-запросы.

    Подробнее →

    Задание 4: TCP Chat

    Реализация многопользовательского чата с использованием протокола TCP и библиотеки threading для максимальных баллов.

    Веб-сервер для обработки GET и POST HTTP-запросов. Сервер принимает и записывает информацию о дисциплине и оценке, отдает информацию обо всех оценках в виде HTML-страницы.
